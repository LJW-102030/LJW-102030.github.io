About / 專案介紹

atttescore is an experimental project for ARM64 program execution and research.

atttescore 是一個用於 ARM64 程式執行與研究的實驗性專案。

Tutorial / 教學

  1. Prerequisites / 前置需求
    • macOS (Intel or Apple Silicon)
    • Clang / Xcode command line tools installed
    • Basic knowledge of ARM64 assembly and Mach-O binaries
  2. Build / 編譯 
    clang src/*.c -Iinclude -o atttescore
  3. Run / 執行 
    ./atttescore test_arm
  4. Example / 範例 
    
.section .text
.global _start
_start:
    mov x0, #42
    ret
    Compile: 
    clang -target arm64-apple-macos12 -nostdlib -Wl,-e,_start -o test_arm test_arm.s
