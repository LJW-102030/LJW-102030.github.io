attescore

Note: This project is still in development. Please use with caution.
注意：本專案仍在開發中，請小心使用。

English:
attescore is an experimental project focused on ARM64 program execution, binary loading, and low-level system research.

中文：
attescore 是一個專注於 ARM64 程式執行、Mach-O 載入與系統層研究的實驗性專案。

Tutorial / 教學

1. Requirements / 前置需求

  • macOS（Intel 或 Apple Silicon）
  • Clang / Xcode Command Line Tools
  • ARM64 Assembly 基本概念

2. Build / 編譯

clang attescore.c arm64_cpu.c memory.c syscall.c -o attescore

3. Run / 執行

./attescore test_arm

4. ARM64 Example / 範例

.global _start
_start:
    mov x0, #7
    ret

clang -target arm64-apple-macos12 -nostdlib -Wl,-e,_start -o test_arm test.s
